# Bachelorarbeit Agent - Schritt-für-Schritt Prozess

## Übersicht
Dieses Dokument definiert alle Schritte, die ein Student bei der Erstellung einer Bachelorarbeit durchläuft. Der Agent wird den User durch jeden dieser Schritte führen.

## Phase 1: Vorbereitung und Planung

### Schritt 1: Themenfindung
- **Ziel**: Ein passendes Thema für die Bachelorarbeit finden
- **Aktivitäten**:
  - Interessen des Students erfragen
  - Fachbereich und Studienrichtung klären
  - Aktuelle Trends und Forschungslücken identifizieren
  - Themenvorschläge generieren und bewerten
  - Themen eingrenzen und präzisieren
- **Output**: 3-5 konkrete Themenvorschläge mit Begründung

### Schritt 2: Exposé erstellen
- **Ziel**: Ein wissenschaftliches Exposé verfassen
- **Aktivitäten**:
  - Forschungsfrage formulieren
  - Relevanz und Begründung der Thematik
  - Aktueller Forschungsstand (erste Literaturrecherche)
  - Methodische Vorgehensweise skizzieren
  - Vorläufige Gliederung erstellen
  - Zeitplan erstellen
- **Output**: Vollständiges Exposé (2-5 Seiten)

### Schritt 3: Betreuung sichern
- **Ziel**: Betreuer/in finden und Thema abstimmen
- **Aktivitäten**:
  - Betreuer/in Kontaktieren
  - Thema präsentieren
  - Feedback einholen
  - Thema anpassen falls nötig
- **Output**: Bestätigung der Betreuung

## Phase 2: Recherche und Konzeption

### Schritt 4: Literaturrecherche
- **Ziel**: Umfassende Literaturrecherche durchführen
- **Aktivitäten**:
  - Suchbegriffe definieren
  - Datenbanken durchsuchen (Google Scholar, PubMed, etc.)
  - Relevante Quellen sammeln und kategorisieren
  - Literaturverwaltung organisieren
  - Quellen bewerten und priorisieren
  - Literaturliste erstellen
- **Output**: Literaturliste mit 30-50 relevanten Quellen

### Schritt 5: Forschungsstand analysieren
- **Ziel**: Den aktuellen Stand der Forschung verstehen
- **Aktivitäten**:
  - Literatur lesen und zusammenfassen
  - Hauptthesen und Argumente identifizieren
  - Forschungslücken identifizieren
  - Theoretischer Rahmen definieren
  - Forschungsfrage präzisieren
- **Output**: Theoretischer Rahmen und präzisierte Forschungsfrage

### Schritt 6: Methodik entwickeln
- **Ziel**: Wissenschaftliche Methodik festlegen
- **Aktivitäten**:
  - Forschungsdesign wählen (qualitativ/quantitativ/mixed)
  - Datenerhebungsmethoden festlegen
  - Analysemethoden bestimmen
  - Ethische Aspekte klären
  - Methodik begründen
- **Output**: Methodenkapitel (erster Entwurf)

## Phase 3: Durchführung

### Schritt 7: Datenerhebung
- **Ziel**: Daten für die Forschung sammeln
- **Aktivitäten**:
  - Erhebungsinstrumente entwickeln (Fragebogen, Interview-Leitfaden, etc.)
  - Teilnehmer/Probanden rekrutieren
  - Daten sammeln
  - Daten dokumentieren
- **Output**: Datensatz

### Schritt 8: Datenanalyse
- **Ziel**: Daten wissenschaftlich auswerten
- **Aktivitäten**:
  - Daten aufbereiten
  - Statistische/qualitative Analyse durchführen
  - Ergebnisse visualisieren
  - Interpretation vorbereiten
- **Output**: Analysierte Daten und erste Ergebnisse

## Phase 4: Schreiben

### Schritt 9: Gliederung finalisieren
- **Ziel**: Finale Struktur der Arbeit festlegen
- **Aktivitäten**:
  - Kapitelstruktur definieren
  - Unterkapitel planen
  - Logischen Aufbau sicherstellen
  - Mit Betreuer/in abstimmen
- **Output**: Finale Gliederung

### Schritt 10: Einleitung schreiben
- **Ziel**: Einleitungskapitel verfassen
- **Aktivitäten**:
  - Problemstellung darstellen
  - Forschungsfrage präsentieren
  - Relevanz begründen
  - Aufbau der Arbeit skizzieren
- **Output**: Einleitung (5-10 Seiten)

### Schritt 11: Theoretischer Teil schreiben
- **Ziel**: Theorie- und Literaturteil verfassen
- **Aktivitäten**:
  - Theoretischen Rahmen darstellen
  - Literatur systematisch aufarbeiten
  - Forschungsstand zusammenfassen
  - Theorie-Modell entwickeln
- **Output**: Theoretischer Teil (15-25 Seiten)

### Schritt 12: Methodik schreiben
- **Ziel**: Methodenkapitel ausformulieren
- **Aktivitäten**:
  - Forschungsdesign beschreiben
  - Methodenwahl begründen
  - Datenerhebung dokumentieren
  - Analysemethoden erklären
- **Output**: Methodenkapitel (5-10 Seiten)

### Schritt 13: Ergebnisse schreiben
- **Ziel**: Ergebnisse präsentieren
- **Aktivitäten**:
  - Ergebnisse strukturiert darstellen
  - Visualisierungen einbinden
  - Deskriptive Darstellung
  - Erste Interpretation
- **Output**: Ergebniskapitel (10-15 Seiten)

### Schritt 14: Diskussion schreiben
- **Ziel**: Ergebnisse diskutieren und interpretieren
- **Aktivitäten**:
  - Ergebnisse interpretieren
  - Mit Literatur vergleichen
  - Limitationen diskutieren
  - Implikationen ableiten
- **Output**: Diskussionskapitel (10-15 Seiten)

### Schritt 15: Fazit schreiben
- **Ziel**: Zusammenfassung und Ausblick
- **Aktivitäten**:
  - Hauptaussagen zusammenfassen
  - Forschungsfrage beantworten
  - Ausblick geben
  - Empfehlungen formulieren
- **Output**: Fazit (3-5 Seiten)

## Phase 5: Finalisierung

### Schritt 16: Überarbeitung
- **Ziel**: Inhaltliche und sprachliche Verbesserung
- **Aktivitäten**:
  - Inhaltliche Logik prüfen
  - Argumentation stärken
  - Sprache verbessern
  - Konsistenz sicherstellen
- **Output**: Überarbeitete Version

### Schritt 17: Korrektur
- **Ziel**: Fehler korrigieren
- **Aktivitäten**:
  - Rechtschreibung prüfen
  - Grammatik korrigieren
  - Zeichensetzung prüfen
  - Formatierung kontrollieren
- **Output**: Korrigierte Version

### Schritt 18: Zitierweise prüfen
- **Ziel**: Zitate und Quellen formal korrekt
- **Aktivitäten**:
  - Zitierstil prüfen (APA, Harvard, etc.)
  - Literaturverzeichnis formatieren
  - Fußnoten/Endnoten prüfen
  - Vollständigkeit sicherstellen
- **Output**: Korrekt zitierte Version

### Schritt 19: Formatierung
- **Ziel**: Formale Anforderungen erfüllen
- **Aktivitäten**:
  - Seitenformat prüfen
  - Schriftart und -größe
  - Zeilenabstand
  - Seitenzahlen
  - Deckblatt
  - Inhaltsverzeichnis
  - Abbildungs- und Tabellenverzeichnis
- **Output**: Formatierte Version

### Schritt 20: Finale Prüfung
- **Ziel**: Letzte Kontrolle vor Abgabe
- **Aktivitäten**:
  - Vollständigkeit prüfen
  - Alle Anforderungen checken
  - Druckversion erstellen
  - PDF generieren
- **Output**: Abgabefertige Version

### Schritt 21: Abgabe
- **Ziel**: Arbeit einreichen
- **Aktivitäten**:
  - Drucken und binden (falls erforderlich)
  - Digital einreichen
  - Bestätigung erhalten
- **Output**: Eingereichte Arbeit

## Agent-Funktionalitäten

### Pro Schritt:
- **Status-Tracking**: Aktueller Schritt wird gespeichert
- **Checklisten**: Konkrete Aufgaben pro Schritt
- **Templates**: Vorlagen für jeden Schritt
- **Feedback**: Agent gibt Feedback zu erstellten Inhalten
- **Erinnerungen**: Wichtige Termine und Deadlines
- **Fortschrittsanzeige**: Visualisierung des Fortschritts

### Übergeordnet:
- **Persistenz**: Fortschritt wird gespeichert
- **Wiederaufnahme**: User kann jederzeit weitermachen
- **Anpassung**: Agent passt sich an individuelle Bedürfnisse an
- **Integration**: Nutzung der bestehenden Features (Editor, Citations, etc.)
